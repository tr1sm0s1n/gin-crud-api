Clone the project


Test the application

```bash
go test
```

Start the application

```bash
go run .
```

```bash
go build -o build/
```

Run the executable

```bash
./build/gin-crud-api
```

For live reload, install Air

```bash
go install github.com/air-verse/air@latest
```

Run the application with Air

```bash
air
```
